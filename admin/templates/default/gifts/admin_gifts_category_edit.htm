修改分类
{#$inputtoken#}
分类名称: 如：2012年国庆礼品卡
有效期开始时间: 生成礼品卡后，从开始时间起才可以使用此类礼品卡充值，如：2012-05-01，0为不限制
有效期截至时间: 到期后此类礼品卡将不能使用，如：2012-05-10，0为不限制
叠加次数: 同会员可用同类礼品卡数，如设为1，每个会员仅能使用一次同类礼品卡，0为不限制，修改此项会影响此分类下的所有礼品卡
赠送积分: 使用礼品卡后获得的积分，修改此项会影响此分类下的所有礼品卡
礼品卡帐号前缀: 如：zhaohulu_ 用于区分不同类型的礼品卡
可用状态: 此类礼品卡是否可用，修改此项会影响此分类下的所有礼品卡
